# 同步服务器分支

### 同步服务器分支
创建自己的分支，不能与服务器上的重复。
创建agou分支，并切换到agou分支
    
    git checkout -b agou
    
将本地分支推送到服务器
    
    git push origin agou
    
将本地分支跟踪服务器分支
    
    git branch --set-upstream-to=origin/分支名称 分支名称
    例：
    git branch --set-upstream-to=origin/agou agou
    
将码云上的dev分支同步到本地，因为开发过程中，所有组员都向这个分支上提交阶段性代码，并从这个分支获取最新代码
    
    git checkout -b dev origin/dev
