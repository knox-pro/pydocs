# 分支合并

### 分支合并
当一个应用开发完成之后，需要合并到dev
1.切换到dev分支
    
    git checkout dev
    
2.获取代码，如果dev分支上有更新的记录则会同步到本地
    
    git pull
    
3.合并
    
    git merge agou
    
4.添加、提交并推送
    
    git add ./
    git commit -m 'agou合并分支'
    git push origin dev
    
5.切换回工作分支
    
    git checkout agou
    
6.同步dev上最新代码，合并到agou分支，继续开发
    
    git merge dev
