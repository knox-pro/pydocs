# 克隆项目

### 克隆项目
添加好ssh公钥之后，回到码云的个人主页\--概览：
![](http://www.chenyangstudy.cn/book-git/img/09.jpg)
可以看到我们刚才创建的项目，点击项目进到项目详细页，点击克隆可以看到有项目地址，将ssh地址复制下来。
![](http://www.chenyangstudy.cn/book-git/img/10.jpg)
将码云仓库中的项目克隆到本地
    
    git clone 项目地址
    
创建一个 django\_web文件夹保存项目
    
    python@python:~$ mkdir django_web
    python@python:~$ cd django_web/
    python@python:~/django_web$ git clone git@gitee.com:cyc5528/django_blog.git
    
第一次连接会提示我们是否连接 输入yes，可以看到刚才在码云上创建的项目文件已经克隆下来。
![](http://www.chenyangstudy.cn/book-git/img/11.jpg)
