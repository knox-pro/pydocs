### 备份

#### 备份数据库
    
    
    mysqldump -uroot -p 数据库名 > python.sql;
    mysqldump -uroot -p python >python.sql
    

提示输入密码，mysql的密码

![](http://cyc.chenyangstudy.cn/book-sql/chapter01/img/32.jpg)

#### 备份某个数据表
    
    
    mysqldump -uroot -p 数据库名  数据表名> class.sql;
    msyqldump -uroot -p python class > class.sql
    

![](http://cyc.chenyangstudy.cn/book-sql/chapter01/img/33.jpg)

#### 恢复

恢复数据库之前要先创建库，我们来看看备份出来的文件，其实就是一堆sql语句，有创建表的语句，但是没有创建库的，所以要先手动创建库，

![](http://cyc.chenyangstudy.cn/book-sql/chapter01/img/34.png)

#### 恢复命令
    
    
    mysql -uroot -p 新数据库名 < python.sql
    

会提示输入数据库的密码
