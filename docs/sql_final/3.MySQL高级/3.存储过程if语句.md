## MySQL中IF语句

语法：
    
    
    IF expression THEN 
       statements;
    END IF;
    

如果表达式(expression)计算结果为TRUE，那么将执行statements语句，否则控制流将传递到END IF之后的下一个语句。

## MySQL IF ELSE语句

如果表达式计算结果为FALSE时执行语句，请使用IF ELSE语句，如下所示：
    
    
    IF expression THEN
       statements;
    ELSE
       else-statements;
    END IF;
    

#### MySQL IF ELSEIF ELSE语句.

如果要基于多个表达式有条件地执行语句，则使用IF ELSEIF ELSE语句如下：
    
    
    IF expression THEN
       statements;
    ELSEIF elseif-expression THEN
       elseif-statements;
    ...
    ELSE
       else-statements;
    END IF;
