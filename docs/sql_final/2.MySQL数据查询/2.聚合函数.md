#### 聚合函数主要是为了快速得到结果，经常使用的几个聚合函数
    
    
    count 统计行数
    max 计算最大值
    min 计算最小值
    sum 求和
    avg  求平均数
    round 保留几位小数
    

#### count 函数

查询学生表中一共有多少人
    
    
    select count(*) from students;
