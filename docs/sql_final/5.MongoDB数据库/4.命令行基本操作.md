#### 终端连接mongodb

直接在终端输入：mongo

出现>表示连接成功

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/11.jpg)

查看当前数据库名称
    
    
    db  #刚连接上mongodb是在test数据库下
    

查看所有数据库名称，列出所有在物理上存在的数据库
    
    
    show dbs
    

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/12.jpg)

切换数据库，如果数据库不存在也并不创建，直到插入数据或创建集合时数据库才被创建
    
    
    use 数据库名称
    

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/13.jpg)

删除当前指向的数据库，如果数据库不存在，则什么也不做
    
    
    db.dropDatabase()
