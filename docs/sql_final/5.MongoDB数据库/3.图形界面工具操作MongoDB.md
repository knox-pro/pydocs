### Robo图形界面操作mongodb

#### 1、创建数据库

点击连接名---右击---选择Create database --- 输入数据库名字。

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/07.png)

#### 2、创建集合

选择Collections--右击---点击create collection---输入集合的名字

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/08.jpg)

#### 操作文档

点击文档的名字右击会出现可操作

view documents 查看文档

insert documents 插入数据

update documents 更新文档信息

remove documents 删除指定文档

remove all documents 删除所有文档

drop collections 删除集合

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/09.jpg)

修改key值内容：

选择key右击--选择edit document编辑
