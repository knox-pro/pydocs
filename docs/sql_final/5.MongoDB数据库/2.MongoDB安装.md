#### ubuntu下安装mongodb

官网地址：<https://www.mongodb.com/download-center>

![](http://cyc.chenyangstudy.cn/book-sql/chapter04/img/01.jpg)

下载tar压缩包

解压
    
    
    tar -zxvf mongodb-linux-x86_64-ubuntu1604-3.4.0.tgz
    

移动到/usr/local/目录下
    
    
    sudo mv -r mongodb-linux-x86_64-ubuntu1604-3.4.0/ /usr/local/mongodb
    

将可执行文件添加到PATH路径中
    
    
    export PATH=/usr/local/mongodb/bin:$PATH
    

启动mongodb:
    
    
    sudo service mongod start
    

停止mongodb：
    
    
    sudo service mongod stop
    

如果上面这种方法无法启动，可以使用下面这中方法,指定配置文件后台运行。
    
    
    sudo mongod -f /etc/mongod.conf  --fork
    

#### 配置

路径在/etc/mongodb.conf

dbpath=/var/lib/mongodb #数据保存路径

systemLog：系统日志

path=/var/log/mongodb/mongodb.log #日志保存路径

logappend=true #开启日志

net：网络配置

bind_ip = 127.0.0.1 # 绑定ip

port = 27017 # 端口号，默认27017
