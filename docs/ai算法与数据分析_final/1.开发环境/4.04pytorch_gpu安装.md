# pytorch_gpu安装

## Windows查看cuda和cudnn
    
    
    #最高支持版本
    nvidia-smi
    #查看 CUDA 版本
    #在命令行中执行： 11.6
    nvcc --version
    #或者进入 CUDA 的安装目录查看：
    C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA
    #查看 cuDNN 版本
    #进入 CUDA 的安装目录查看文件 cudnn_version.h ：
    cd C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.6\include
    cat C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.6\include\cudnn_version.h
    

## 我安装的版本
    
    
    CUDA 11.6
    CUDNN 8.4.0.27、CUDNN 8.9.0.131
    

## 创建虚拟环境python3.8
    
    
    conda create -n py38_torch python=3.8
    conda activate py38_torch
    

## GPU的torch版本车轮下载

官方地址
    
    
    https://download.pytorch.org/whl/torch_stable.html
    

ctrl+F，搜素 cu116

cu116/torch-1.13.1%2Bcu116-cp38-cp38-win_amd64.whl cu116/torchvision-0.14.1%2Bcu116-cp38-cp38-win_amd64.whl 
    
    
    pip install "torch-1.13.0+cu116-cp38-cp38-win_amd64.whl"
    

## torch和torchvision版本对应关系
    
    
    #torch1.13.0对应torchvision0.14.0
    pip install "torchvision-0.14.0+cu116-cp38-cp38-win_amd64.whl"
    

## CPU版本用pip安装torch即可
    
    
    pip install torch
