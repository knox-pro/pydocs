# linux虚拟环境python

## conda构建虚拟环境（推荐）

### 1.下载Minicondaxxx.sh
    
    
    https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/?C=M&O;=D
    

### 2.安装conda
    
    
    bash Miniconda3xxxx.sh
        点击继续:回车
        用户协议是否同意：yes
        是否改路径 ：/cychome/miniconda3
        是否将安装位置添加环境变量(.bashrc)：yes
    安装完成:
    bash
    直接进入conda环境，在前面会出现一个base.
    

没有添加环境变量

  * 打开设置文件


`vim ~/.bashrc`

  * 在打开的文件末添加


`export PATH="/cychome/miniconda3/bin:$PATH"`

  * 使文件配置立即生效


`source ~/.bashrc`

### 3.conda简单操作
    
    
    #查看安装位置
    which conda
    
    #进入conda环境，在前面会出现一个base
    bash
    
    # 退出虚拟环境
    conda deactivate 
    # 或者 source deactivate
    
    #管理环境
    conda env list
    
    # 创建python3.6环境
    conda create --name py36_conda python=3.6
    
    # 激活环境
    source activate py36
    
    # 导出环境安装包
    conda env export > py36.yaml
    
    # 利用导出的conda环境安装包建立同样的虚拟环境
    conda env create -f py36.yaml 
    
    # 删除环境
    conda remove -n py36 --all
    
    # 查看已有的虚拟环境
     conda info --envs
    

## virtualenv虚拟环境（不推荐）

### 1.pip安装virtualenv虚拟环境
    
    
    #用source启动  virtualenv创建
    pip install virtualenv
    virtualenv 环境名
    source 环境地址/activate
    
    
    
    #用workon启动  mkvirtualenv创建
    pip install virtualenvwrapper
    添加路径
    vim /usr/local/bin/virtualenvwrapper.sh
    source /usr/local/bin/virtualenvwrapper.sh
    source ~.xxxrc
    mkvirtualenv 环境名
    

### 2.pyenv进行python版本管理
    
    
    可安装不同版本python，可实现不同python中的虚拟环境切换。
    

安装
    
    
    curl https://pyenv.run | bash
    
    rm -rf /root/.pyenv
    curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash
    

添加环境变量
    
    
    export PATH=$HOME/.pyenv/bin:$PATH
    eval "$(pyenv init -)"
    eval "$(pyenv virtualenv-init -)"
    

  * 查看本机安装版本


pyenv versions

*表示当前使用的版本

  * 查看可安装的版本


pyenv install -l

  * 安装一个python版本


pyenv install 3.6.4

pyenv uninstall 3.6.4

  * 切换版本


pyenv shell 3.6.4

shell > local>global。我们可以用shell来管理不同的python版本。

### 3.pyenv-virtualenv来管理虚拟环境

ls -la ~/.pyenv/plugins 查看插件可以看到pyenv-virtualenv

列出当前python3.6.4环境中的虚拟环境
    
    
    pyenv virtualenvs
    

创建虚拟环境（不指定python版本则使用当前python环境的版本，指定则使用指定版本,但必须是已经安装的python版本）
    
    
    pyenv virtualenv 3.6.4 虚拟环境名
    
    pyenv virtualenv 虚拟环境名  #使用当前所在的python版本创建虚拟环境
    

激活当前python版本的虚拟环境
    
    
    pyenv activate 虚拟环境名
    

## virtualenvwrapper构建虚拟环境
    
    
    #使用pip安装包前，先更新pip 
    pip3 install --upgrade pip
    #安装virtualenv、virtualenvwrapper 
    pip3 install virtualenv
    pip3 install virtualenvwrapper
    

### 1.配置
    
    
    mkdir $HOME/.virtualenvs
    

### 2.使用vim编辑.bashrc文件
    
    
    vim  ~/.bashrc
    

在文末填入以下代码并保存,注意virtualenvwrapper.sh的位置 
    
    
    cd /
    find -name virtualenvwrapper.sh
    
    
    
    VIRTUALENVWRAPPER_PYTHON=/usr/bin/python3    # 指定virtualenvwrapper执行的python版本
    export WORKON_HOME=$HOME/.virtualenvs    # 指定虚拟环境存放目录，.virtualenvs目录名可自拟
    source /usr/local/bin/virtualenvwrapper.sh    # virtualenvwrapper.sh所在目录
    

### 3.运行.bashrc文件**
    
    
    source ~/.bashrc
    

如果source ~/.bashrc 报错,修改/usr/local/bin/virtualenvwrapper.sh 第50行 
    
    
    vim /usr/local/bin/virtualenvwrapper.sh 
    VIRTUALENVWRAPPER_PYTHON="$(command \which python3)"
    

### 4.创建虚拟环境
    
    
    mkvirtualenv -p /usr/bin/python3 py3-yt
    #退出虚拟环境：
    deactivate
    #进入虚拟环境：
    workon py3-yt
