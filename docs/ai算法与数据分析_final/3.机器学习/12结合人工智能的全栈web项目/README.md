# 结合人工智能的全栈web项目

![](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/12结合人工智能的全栈web项目/2.jpg)

## 1、 介绍

  * 基于django后端只是,html，js，css前端只是，机器学习，神经网络等人工智能知识，完成一个综合性项目。


![](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/12结合人工智能的全栈web项目/1.jpg)

## 2、 主要内容。

  * 1、 基于线性回归模型完成广州房价预测页面功能
  * 2、 基于十大机器学习模型完成泰坦尼克号获救预测页面功能
  * 3、 实现深度学习模型完成动物图像分类页面功能
  * 4、 实现深度学习模型完成人脸识别页面功能
  * 5、 实现深度学习模型完成手写数字识别
