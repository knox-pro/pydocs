# 神经网络

> 目前，深度学习 (Deep Learning，简称DL) 在算法领域可谓是大红大紫，现在不只是互联网、人工智能，生活中的各大领域都能反映出深度学习引领的巨大变革。要学习深度学习，那么首先要熟悉神经网络 (Neural Networks，简称NN) 的一些基本概念。
> 
> 当然，这里所说的神经网络不是生物学的神经网络，我们将其称之为人工神经网络 (Artificial Neural Networks，简称ANN) 貌似更为合理。神经网络最早是人工智能领域的一种算法或者说是模型，目前神经网络已经发展成为一类多学科交叉的学科领域，它也随着深度学习取得的进展重新受到重视和推崇。 神经网络方面的研究很早就已出现，今天“神经网络”已是一个相当大的、多学科交叉的学科领域。神经网络中最基本的成分是神经元模型。

## 1 感知器

为了理解神经网络，我们应该先理解神经网络的组成单元——神经元。神经元也叫做感知器。还记得之前的线性回归模型中权重的作用吗？ 每一个输入值与对应权重的乘积之和得到的数据或通过激活函数来进行判别。下面我们看一下感知器：

![img_1.png](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/09神经网络与OCR字符识别/img_1.png)

可以看到，一个感知器有如下组成部分： \- 输入权值，一个感知器可以有多个输入x1,x2,x3...xn每个输入上有一个权值wi

  * 激活函数，感知器的激活函数有许多选择

  * 输出，y=f(w∗x+b)


我们了解过sigmoid函数是这样,在之前的线性回归中它对于 二类分类 问题非常擅长。所以在后续的多分类问题中，我们会用到其它的激活函数

## 2 神经网络

那么我们继续往后看，神经网络是啥？

![img_3.png](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/09神经网络与OCR字符识别/img_3.png)

神经网络其实就是按照一定规则连接起来的多个神经元。 \- 输入向量的维度和输入层神经元个数相同 \- 第N层的神经元与第N-1层的所有神经元连接，也叫 全连接 \- 上图网络中最左边的层叫做输入层，负责接收输入数据；最右边的层叫输出层，可以有多个输出层。我们可以从这层获取神经网络输出数据。输入层和输出层之间的层叫做隐藏层，因为它们对于外部来说是不可见的。 \- 而且同一层的神经元之间没有连接 \- 并且每个连接都有一个权值，

那么我们以下面的例子来看一看，图上已经标注了各种输入、权重信息。

![img_4.png](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/09神经网络与OCR字符识别/img_4.png)

对于每一个样本来说，我们可以得到输入值x1,x2,x3,,也就是节点1，2，3的输入值，那么对于隐层每一个神经元来说都对应有一个偏置项b，它和权重一起才是一个完整的线性组合

![img_5.png](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/09神经网络与OCR字符识别/img_5.png)

这样得出隐层的输出，也就是输出层的输入值. 矩阵表示

![img_6.png](http://cyc.chenyangstudy.cn/book_chenyang/a02机器学习/09神经网络与OCR字符识别/img_6.png)

> 同样，对于输出层来说我们已经得到了隐层的值，可以通过同样的操作得到输出层的值。那么重要的一点是，分类问题的类别个数决定了你的输出层的神经元个数

## 3.案例演示

### 1、神经元

权重：过滤输入信息，针对不同的数据，提高或者降低其作用和影响(反向传播更新权重) w

偏置：没有任何输入时的输出。 b

激励（活）函数

### 2、层

每一层可以由1到n个神经元组成，居中的神经元接收上一层的输出，并且为下一层提供输入，数据只能在相邻之间传递，

不能跨层传输。

输入层：接收收入样本的各个特征

隐藏层：0到多个，通过权重，偏置和激活函数，对所接收到的来自上一层的数据进行计算。 out = f(np.dot(X,W)+b)

输出层：功能与隐藏层相同，将计算结果作为输出的每一个特征。

**若隐藏层数多于一层，则可以被称为深度神经网络，通常使用的深度神经网络，其隐藏层可以多达数十层到上百层。基于这样的

结构学习模型被称为深度学习。

### 3、最简单的神经网络：感知器

示例代码：01-neuron.py 
    
    
    import numpy as np
    import matplotlib.pyplot as mp
    import neurolab as nl #神经网络模块 pip install neurolab
    #构造数据
    x = np.array([
        [0.3,0.2],
        [0.1,0.4],
        [0.4,0.6],
        [0.9,0.5],
    ])
    y = np.array([
        [0],
        [0],
        [0],
        [1]
    ])
    #构造单层输出神经网络模型
    model = nl.net.newp([[0,1],[0,1]],1)
    #模型训练
    #定义误查
    error = model.train(x,y,epochs=10,show=1,lr=0.01) #学习速率0.01
    #模型预测
    #生成网格点
    l,r,h = x[:,0].min()-1,x[:,0].max()+1,0.01
    b,t,v = x[:,1].min()-1,x[:,1].max()+1,0.01
    grid_x = np.meshgrid(np.arange(l,r,h),np.arange(b,t,v))
    flat_x = np.c_[grid_x[0].ravel(),grid_x[1].ravel()]
    flat_y = model.sim(flat_x)   #todo；模型预测
    grid_y = flat_y.reshape(grid_x[0].shape)
    #可视化
    
    mp.figure("Neuron",facecolor="lightgray")
    mp.title("Neuron",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.pcolormesh(grid_x[0],grid_x[1],grid_y,cmap="gray")
    mp.scatter(x[:,0],x[:,1],c=y.reshape(-1),cmap="brg",s=60)  #画样本点
    mp.show()
    
    #训练过程
    mp.figure("Training Process",facecolor="lightgray")
    mp.title("Process",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.grid(linestyle=":")
    mp.plot(error,'o-',c="dodgerblue",linewidth=2)
    mp.show()
    

### 4、单层多输出神经网络

示例代码：02-mono.py 
    
    
    import numpy as np
    import matplotlib.pyplot as mp
    import neurolab as nl #神经网络模块 pip install neurolab
    #构造数据
    data = np.loadtxt('mono.txt')
    x = data[:,:2]
    y = data[:,-2:]
    
    #对标签进行字符串拼接处理:"0.1","1.0"，’1.1‘，’0.0‘
    train_labels = []  #获取分类数
    for train_row in y:
        train_row = train_row.astype(int).astype(str)
        train_labels.append('.'.join(train_row))
    #把标签位置限定好，预测结果要和测试结果保持一致,去重并升序
    label_set = np.unique(train_labels)
    
    #位置0，1,2,3
    train_codes = []
    for train_label in train_labels:
        train_code = np.where(label_set==train_label)[0][0]  #位置：0，1,2,3
        train_codes.append(train_code)
    train_codes = np.array(train_codes)
    # print(train_codes)
    
    #todo：构造单层多输出神经网络模型
    model = nl.net.newp([[x[:,0].min(),x[:,0].max()],[x[:,1].min(),x[:,1].max()]],2)   #输出层2个神经元做4分类问题
    #模型训练
    #定义误查
    error = model.train(x,y,epochs=10,show=1,lr=0.01) #学习速率0.01
    #模型预测
    #生成网格点
    l,r,h = x[:,0].min()-1,x[:,0].max()+1,0.05
    b,t,v = x[:,1].min()-1,x[:,1].max()+1,0.05
    grid_x = np.meshgrid(np.arange(l,r,h),np.arange(b,t,v))
    flat_x = np.c_[grid_x[0].ravel(),grid_x[1].ravel()]
    flat_y = model.sim(flat_x)   #todo；模型预测 ,返回的二阶矩阵，【0,1】
    flat_ys = []
    #对标签进行字符串拼接处理:"0.1","1.0"，’1.1‘，’0.0‘
    train_labels = []  #获取分类数
    for y in flat_y:
        y = y.astype(int).astype(str)
        flat_ys.append('.'.join(y))
    #位置0，1,2,3
    grid_ys = []
    for y in flat_ys:
        code = np.where(label_set==y)[0][0]  #位置：0，1,2,3
        grid_ys.append(code)
    grid_ys = np.array(grid_ys).ravel()
    grid_y = grid_ys.reshape(grid_x[0].shape)
    #可视化
    
    mp.figure("Neuron",facecolor="lightgray")
    mp.title("Mono Neuron",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.pcolormesh(grid_x[0],grid_x[1],grid_y,cmap="gray")
    mp.scatter(x[:,0],x[:,1],c=train_codes,cmap="brg",s=60)  #画样本点
    mp.show()
    
    #训练过程
    mp.figure("Training Process",facecolor="lightgray")
    mp.title("Process",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.grid(linestyle=":")
    mp.plot(error,'o-',c="dodgerblue",linewidth=2)
    mp.show()
    

### 5、深度神经网络

示例代码：03-deep.py 
    
    
    import numpy as np
    import neurolab as nl
    import matplotlib.pyplot as mp
    
    
    #1、构造回归数据
    x = np.linspace(-10,10,100)
    y = 2*np.square(x) + 1
    #标签集标准化
    y /= np.linalg.norm(y)
    x = x.reshape(-1,1)
    y = y.reshape(-1,1)
    #todo:2、构造深度神经网络模型
    model = nl.net.newff([[x.min(),x.max()]],[10,10,1]) #两个隐藏层，每层10个神经元
    # model.trainf = nl.train.train_gd #设置为梯度下降
    #3、模型训练
    error = model.train(x,y,epochs=1000,show=10,goal=0.01) #训练目标最下误差0.01
    #4、模型预测
    test_x = np.linspace(-10,10,1000)
    test_x = test_x.reshape(-1,1)
    pred_y = model.sim(test_x)
    #5、可视化
    mp.figure("Neuron",facecolor="lightgray")
    mp.title("Deep Neuron",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.scatter(x,y,c="dodgerblue",s=60,label="train",alpha=0.75)  #画样本点
    mp.plot(test_x,pred_y,'-',c="green",label="model") #画模型
    mp.legend()
    mp.show()
    
    #训练过程
    mp.figure("Training Process",facecolor="lightgray")
    mp.title("Process",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.grid(linestyle=":")
    mp.plot(error,'o-',c="dodgerblue",linewidth=2)
    mp.show()
    

### 6、OCR字符识别

示例代码：04-acrdb.py 
    
    
    import numpy as np
    import cv2 as cv
    #16*8图片，总共128个特征
    
    with open("ocrdb.dat","r") as f:
        for line in f.readlines():
            items = line.split('\t')
            char = items[1]
            image = items[6:-1]
            image = np.array(image,dtype=np.uint8)
            image *= 255
            image = image.reshape(16,8)
            image = cv.resize(image,None,fx=25,fy=25) #将图片横向纵向均放大25倍
            cv.imshow(char,image) #图名，图像矩阵
    
            if cv.waitKey(100) == 27:
                break
    
    
    
    import numpy as np
    import matplotlib.pyplot as mp
    import neurolab as nl
    import sklearn.model_selection as ms
    
    #1/构造数据
    
    charset = "chenyang"
    x,y = [],[]
    with open("ocrdb.dat","r") as f:
        for line in f.readlines():
            items = line.split('\t')
            char = items[1]
            image = items[6:-1]
            if char in charset:
                code = np.zeros(len(charset),dtype=int)  #[0 0 0 0 0 0 0]
                #独热编码
                code[charset.index(char)] = 1
                y.append(code)
                x.append(np.array(image,dtype=int))
                if len(x) >= 50:
                    break
    x = np.array(x)
    y = np.array(y)
    #训练集，测试集切分
    train_x,test_x,train_y,test_y = ms.train_test_split(x,y,test_size=0.2,random_state=8)
    #2、构造模型
    #值域,128个特征
    input_range = []
    for i in x.T:
        input_range.append([0,1])
    model = nl.net.newff(input_range,[128,16,y.shape[1]])
    model.trainf = nl.train.train_gd
    #3、模型训练
    erro = model.train(train_x,train_y,epochs=10000,show=10,goal=0.01) #训练目标最小误差0.01
    #4、模型预测
    pred_test_y = model.sim(test_x)
    print(pred_test_y)
    # for y,y_ in zip(test_y,pred_test_y):
    #     print(f"预测结果：{y_}----》真实结果{y}")
    def decode(codes):
        return ''.join(charset[i.argmax()] for i in codes)
    
    true_string = decode(test_y)
    pred_string = decode(pred_test_y)
    print(f"预测结果：{pred_string}----》真实结果{true_string}")
    #5、可视化
    axes = mp.subplots(1,len(test_x),num="OCR",facecolor="lightgray")[1]
    for ax,char_img,true_char,pred_char in zip(axes,test_x,true_string,pred_string):
        ax.matshow(char_img.reshape(16,8),cmap="brg")
        ax.set_title(f"{true_char}{'==' if true_char==pred_char else '!='}{pred_char}",
                     color="green" if true_char==pred_char else "red",fontsize=14)
    
        ax.set_xticks(())
        ax.set_yticks(())
    
    mp.show()
    
    
    
    #训练过程
    mp.figure("Training Process",facecolor="lightgray")
    mp.title("Process",fontsize=20)
    mp.xlabel("X",fontsize=14)
    mp.ylabel("Y",fontsize=14)
    mp.tick_params(labelsize=10)
    mp.grid(linestyle=":")
    mp.plot(erro,'o-',c="dodgerblue",linewidth=2)
    mp.show()
