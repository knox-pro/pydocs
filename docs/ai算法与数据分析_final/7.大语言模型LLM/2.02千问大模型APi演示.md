## LangChain简介

### LangChain框架

大模型一般有两种形态“呈现”在我们眼前，一种是训练好的那种二进制文件，另外一种是将大模型的二进制文件进行部署之后暴露出一些相应的接口，但是无论是那种形式， LLM 只提供了一个非常基础的调用方式，当我们要构建一个复杂的 Chat Bot 时，就需要考虑如何保存聊天的上下文、如何进行网络检索、如何加载本地数据、如何便捷管理 Prompt 等等工程问题。

甚至是当我们切换到不同的 LLM 时，模型的输入和输出结构差异都非常巨大，微量的需求就要修改大量的代码，或者在业务代码中做大量的判断与识别， 让代码可维护性极差，但是其实不同 LLM 的交互流程其实都非常接近，可以看成是一个基础聊天机器人的链条，传入提示词，输出对应的结果。

AI 应用开发框架应运而生，其中最热门、更新速度最快、最稳定的框架就是 LangChain，而且目前 LangChain 提供了 Python 和 JavaScript 两个版本，适配了当前 AI 环境下最热门的两种语言。

  * LangChain-Python：https://github.com/langchain-ai/langchain
  * LangChain-JavaScript：https://github.com/langchain-ai/langchainjs

## 基于langchain实现大模型问答

### 提示词模板示例
    
    
    import os
    from langchain_community.llms import Tongyi #Tongyi（通义） 大语言模型的接口
    from langchain_core.prompts import PromptTemplate #定义和管理提示词模板的核心类
    
    os.environ["DASHSCOPE_API_KEY"] = "sk-xxx" #改成自己的
    
    llm = Tongyi()
    
    prompt = PromptTemplate.from_template(
        "问题：{question}\n回答：让我们一步一步思考。"
    )
    chain = prompt | llm
    question = "中国篮球运动员，在NBA成就最高的球员是谁?"
    
    # 调用链并打印结果
    result = chain.invoke({"question": question})
    print("答案:", result)
    

结果：
    
    
    答案: 我们来一步一步思考这个问题：
    
    第一步：明确问题的关键词  
    问题问的是“中国篮球运动员”中，在“NBA成就最高的球员”是谁。  
    这意味着我们需要关注以下几点：  
    - 球员必须是中国籍（或代表中国）；  
    - 曾在NBA打球；  
    - 在NBA取得的成就是衡量标准，比如出场次数、数据、影响力、荣誉等。
    
    第二步：回顾曾进入NBA的中国球员有哪些？  
    历史上，有几位中国球员曾进入NBA，包括：  
    1. 王治郅（Wang Zhizhi）——2001年加盟达拉斯小牛队（现独行侠），首位登陆NBA的中国球员。  
    2. 巴特尔（Battur）——2001年进入NBA，曾效力掘金、马刺等队，2003年随圣安东尼奥马刺队获得NBA总冠军（虽然出场有限）。  
    3. 姚明（Yao Ming）——2002年以状元秀身份被休斯顿火箭队选中，效力NBA 8个赛季（因伤提前退役）。  
    4. 孙悦——2007年被湖人队选中，2008年短暂效力湖人，获得过NBA总冠军（但出场极少）。  
    5. 易建联（Yi Jianlian）——2007年第六顺位被雄鹿队选中，先后效力雄鹿、篮网、奇才、独行侠等队，共5个赛季。  
    6. 崔永熙——2024年与布鲁克林篮网签约，目前刚进入NBA，尚无显著成就。
    
    第三步：比较这些球员在NBA的成就  
    - 王治郅：开创历史，但数据和出场时间有限。  
    - 巴特尔：获得NBA总冠军戒指，但角色非常边缘。  
    - 孙悦：也获得总冠军，但几乎未出场，贡献极小。  
    - 易建联：有一定表现，最高单场31分，但未能稳定立足。  
    - 姚明：  
      - 8次入选NBA全明星（其中多次为球迷票选首发中锋）；  
      - 5次入选NBA最佳阵容（2次二阵，3次三阵）；  
      - 场均19.0分，9.2篮板，1.9盖帽，投篮命中率高达52.4%；  
      - 长期担任火箭队核心，带领球队多次进入季后赛；  
      - 2016年入选奈·史密斯篮球名人堂，是唯一获此殊荣的中国球员；  
      - 对NBA全球化，尤其是中国市场影响巨大。
    
    第四步：综合判断“成就最高”的标准  
    如果从个人数据、荣誉、影响力、稳定性等多个维度来看，姚明在NBA的表现远超其他中国球员。他是真正意义上在NBA成为超级巨星的中国球员。
    
    第五步：得出结论  
    因此，在所有中国篮球运动员中，在NBA成就最高的球员是姚明。

### 基于langchain构建agent智能体
    
    
    想象成一个工厂的生产线：
        agent = 大脑（制定计划）
        tools = 工具库（执行能力）
        AgentExecutor = 生产线经理（协调大脑和工具，确保任务完成）
    

示例代码
    
    
    from langchain_community.llms import Tongyi #Tongyi（通义） 大语言模型的接口
    from langchain.agents import tool
    from langchain import hub
    from langchain.agents import AgentExecutor,create_json_chat_agent
    import os
    os.environ["DASHSCOPE_API_KEY"] = "sk-xxx" #改成自己的
    
    @tool
    def get_word_length(word: str) -> int:
        """Returns the length of a word."""
        return len(word)
    
    # 使用agent
    llm = Tongyi() #Tongyi（通义） 大语言模型的接口
    tools = [get_word_length]
    #拉取一个名为 "hwchase17/react-chat-json" 的提示词模板。hwchase17是模板作者,react-chat-json是模板名称
    prompt = hub.pull("hwchase17/react-chat-json")
    agent = create_json_chat_agent(llm, tools, prompt) #创建能够输出规范 JSON 格式响应的聊天智能体
    # AgentExecutor（代理执行器），它是 LangChain 中负责实际运行代理的核心组件。
    agent_executor = AgentExecutor(agent=agent, tools=tools, verbose=True,handle_parsing_errors=True) #verbose=True 时，invoke后会打印相关输出
    #agent_executor.invoke() 是 执行代理任务的核心方法，它负责启动整个代理的工作流程。
    agent_executor.invoke({"input": "我想提高自己的篮球投篮命中率，有什么建议?"})
